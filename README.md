# SmartTask
Приложение для создания задач и целей наподобии TODOIST\JIRA
# Задача
Тренировка C++ и QT
# Цель
Создание приложения которое сможет конкурировать по функционалу с конкурентами, а конкретнее:
- Возможность иметь учетную запись
- Создание канбанов и задач
- Кастомизация приложения
- Веб версия и настольное приложение
